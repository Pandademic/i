## `write x`
Write `x` to stdout.

## `rewrite x`
Write `'\r'..x` to stdout.

## `print x`
Write `x..'\n'` to stdout.

## `reprint x`
Write `'\r'..x..'\n'` to stdout.

## `error x`
Write `x` to stderr.

## `reerror x`
Write `'\r'..x` to stderr.

## `printerror x`
Write `x..'\n'` to stderr.

## `reprinterror x`
Write `'\r'..x..'\n'` to stderr.

## `input`
A line from the standard input, without the newline.
