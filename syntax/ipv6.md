# IPv6 Packets

IPv6 packets are mutable.

## `ipv6packet`
The IPv6 packet type.

## `ipv6packet c`
An IPv6 packet from the collection, `c`.

## `p.class`
`p`'s traffic class.

## `p.label`
`p`'s flow label.

## `p.payload`
`p`'s payload.

## `#p.payload`
`p`'s payload length in bytes.

## `p.nextheader`
`p`'s Next Header.

## `p.hoplimit`
`p`'s hop limit.

## `p.srcaddr`
`p`'s source address.

## `p.destaddr`
`p`'s destination address.

## `p.payload`
`p`'s payload.

# IPv6 Addresses

## `ipv6addr`
The IPv6 address type.

## `ipv6addr c`
An IPv6 address from the collection, `c`.

## `a#i`
The `i`th of the 4 divided numbers of the address, `a`.

## `a#r`
An array of the divided numbers of `a` in range, `r`.

## `a#i = x`
Set `a#i` to `x`.

## `a#r = x`
Set `a#r` to the array, `x`.
