## `progress`
The progress type, or a progress with the default format.

## `progress f`
A progress with the format, `f`: TODO

## `p` as `bool`
Whether the progress, `p`, is ongoing.

## `p x outof y`
Start `p` with a total of `x`, and a current value of `y`.

## `p at n`
Set `p`'s current value to `n`.

## `p x`
`p 0 outof x`.
